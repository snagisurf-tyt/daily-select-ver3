# バグ修正設計: 17時以降の縦の罫線が消えている

## 現状の問題点
現在のCSS定義では、`.day-column` に対して `border-right: 1px solid var(--border);` が設定されているが、特定の条件下（17時以降）でこのスタイルが適用されていない。

## 根本原因
CSSのセレクタの優先順位が不十分であることが原因。特定の条件下で他のスタイル定義が優先され、罫線が表示されなくなっている。

## 修正方案
CSSのセレクタの優先順位を明確にするために、以下のようにセレクタを修正する:

```css
.days-container .day-column {
    border-right: 1px solid var(--border) !important;
}

.days-container .day-column:last-child {
    border-right: none !important;
}
```

この修正により、すべての `.day-column` 要素に対して一貫したスタイルが適用され、17時以降の時間帯でも縦の罫線が正しく表示される。

## 影響範囲
この修正はカレンダーの表示にのみ影響し、他の機能には影響を与えない。

## 検証方法
1. 修正後のアプリケーションを開く
2. すべての時間帯（特に17時以降）で縦の罫線が正しく表示されていることを確認する
3. 異なるブラウザで表示を確認し、互換性があることを確認する
